1. Python Basics

Writing First Python Program

What is it?

The print() function is the most basic tool in Python. It tells the computer to display information on the screen.

How it Works (Step-by-Step)

Step 1: Type the command print (always lowercase!).
Step 2: Add ( ) parentheses to hold your message.
Step 3: Use " " quotes if you are printing text.
Example 1:
print("Hello Students")
Line 1: print tells Python to display something.
Line 2: "Hello Students" is the actual text to show.
Hello Students

Business Use Case

In a bank app, when you check your balance, the app uses a print-like function to show: "Your Balance is $5,000".

Wrong Print("Hello")

Always use lowercase print.

Parentheses ( ) are mandatory.

2. Memory Basics

Variables

What are Variables?

Variables are containers used to save data. Once you store a value in a variable, you can use that name later in your code instead of typing the value again.

How to Create a Variable

name = "John"

1. The Name: name (The label on the box).
2. The Equal Sign: = (Used to put the item inside).
3. The Value: "John" (The actual data inside).
Example:
age = 25
print(age)
Line 1: We stored the number 25 in a box named age.
Line 2: We asked Python to find the box age and show what's inside.
25

Practical Use Case (Business)

Imagine you manage a store. You can store the price of a product in a variable:

iphone_price = 999
Wrong "age" = 25

Variable names should NOT have quotes.

Correct age = 25

Just write the name directly!

Quick Revision
Variable = Box for data.
Use = to assign a value.
Call the variable name to use the data.

3. Naming Rules

Rules of Variables

Why do we need rules?

Python is strict about how you name your variables. Following these rules prevents "Syntax Errors" which stop your program from running.

The 3 Golden Rules

1. Letters/Numbers/Underscore: Names can have a-z, 0-9, and _.
2. NO starting with Digits: 1name is illegal. name1 is legal.
3. Case Sensitive: Age and age are different boxes!
Example:
user_id = 101 # Valid
price_2 = 500 # Valid
Invalid Names 2nd_user = "B"
user name = "B"

Can't start with numbers or have spaces.

Valid Names user2 = "B"
user_name = "B"

Numbers at end and underscores are perfect!

Quick Revision
Use _ instead of spaces.
Never start with a number.
Name is NOT the same as name.

5. Text Data

Python Strings

What is a String?

A string is any piece of text. In Python, anything wrapped in quotes ("" or '') is treated as a string.

Core Rules

Python doesn't care if you use 'Double' or "Single" quotes, as long as you match them!

1. Single: 'Hello'
2. Double: "Hello"
3. Multi-line: Use triple quotes """ """ for long paragraphs.
Example:
shop = "Python Mart"
print(shop)
Quotes tell Python: "Don't treat this as code, just treat it as text."
Python Mart
Unmatched name = "John'

Starting with " and ending with ' causes an error.

Perfect Match name = "John"

The start and end quotes must be the same!

Quick Revision
Strings represent text.
Always wrap text in " " or ' '.
Spaces inside quotes are also part of the string.

6. Part of Text

String Slicing

What is String Slicing?

Sometimes you don't need the whole text—just a piece of it. Slicing lets you cut a string at specific points.

Important: The Index Rule

In Python, we count from **0**, not 1.
"P Y T H O N"
0 1 2 3 4 5

Formula: string[start : end]
- Start: The first character you want (inclusive).
- End: The character where you STOP (exclusive - it's not included!).
Example:
word = "PYTHON"
print(word[0:2])
It starts at P (0) and stops BEFORE T (2). So it gives P and Y.
PY
Off by One word[1:3]

Beginners often think [1] is the first letter. In Python [0] is the first!

Correct word[0:3]

Starts at 0 to get the very first letter.

Quick Revision
Always start counting from 0.
The 'end' index is NOT included in result.
[:4] means from start to 4.

7. Joining Text

String Concatenation

What is Concatenation?

Concatenation is just a technical word for **Joining** or **Gluing** strings together using the + sign.

Real-Life Analogy: Think of Train Cars. On their own, they are separate, but you can hook them together to make one long train.

How to Glue Strings

full = "Hello" + "World"
Result: HelloWorld (Wait! No space?)
Correct Example:
first = "John"
last = "Doe"
print(first + " " + last)
We added a manual space " " in the middle.
John Doe

Practical Use Case (E-commerce)

When an online store prints your delivery label, they join First Name + Space + Last Name.

Missing Space "Hi"+"User"

Python won't add a space automatically. Result: "HiUser".

Double Glue "Hi" + " " + "User"

Explicitly adding the space makes it readable.

Quick Revision
Use + to join strings.
Python never adds spaces for you.
You can join variables and literal text.

8. Type Conversion

Type Casting

What is Type Casting?

Sometimes Python sees data as text, but you need it to be a number. Casting is the process of manually forcing a value into a new type.

Common Conversion Tools

1. int(): Turns data into a whole number.
2. float(): Turns data into a decimal.
3. str(): Turns data into text.
Example (Business):
price = "99"     # Python sees this as TEXT
total = int(price) + 1 # Convert to NUMBER 99
Without int(), Python would try to add text to a number (Error!).
100
Invalid Cast int("Hello")

You can only cast text that looks like a number. "Hello" cannot be a number.

Valid Cast int("100")

Since "100" looks like a number, Python converts it perfectly.

Quick Revision
int() for whole numbers.
float() for decimals.
Input from users is ALWAYS a string by default.

Python Lists [ ]

What is a List?

A List is an ordered collection of items. It allows you to store multiple values in a single variable and modify them as needed.

1. Brackets: Always use [ ] square brackets.
2. Changeable: You can use .append() to add new items.
Example:
fruits = ["Apple", "Banana"]
fruits.append("Cherry")
print(fruits)
Line 2: .append() adds "Cherry" to the end of the list.
['Apple', 'Banana', 'Cherry']
List Summary
Uses [ ] square brackets.
.append() adds items.

Python Tuples ( )

What is a Tuple?

A Tuple is a collection of items that is immutable, meaning it cannot be changed once it is created.

1. Brackets: Always use ( ) parentheses.
2. Safety: Because it can't change, it's safer for sensitive data.
Example:
location = (12.97, 77.59) # GPS points
print(location[0])
12.97

Python Sets { }

What is a Set?

A Set is an unordered collection of items where every element is unique. It automatically handles duplicate removal.

Example:
numbers = {1, 2, 2, 3}
print(numbers)
Note: The second '2' was automatically removed!
{1, 2, 3}
